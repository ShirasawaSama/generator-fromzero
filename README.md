## 使用

首先, 请使用 [npm](https://www.npmjs.com/) 安装 [Yeoman](http://yeoman.io) 及 `generator-fromzero` 模块 **(请使用下面的安装命令)**

```bash
npm install -g yo
npm install -g generator-fromzero
```

接着, 输入以下指令快速创建项目, 您可以按照提示操作:

```bash
yo fromzero
```
